# Contributing to Ghost-Web-Scanner

Terima kasih sudah ingin berkontribusi! Silakan ikuti panduan singkat berikut.

## Langkah kontribusi

1. Fork repository ini.
2. Clone ke lokal:
   ```bash
   git clone https://github.com/<username>/Ghost-Web_Scanner.git
   cd Ghost-Web-scanner
   ```
3. Buat branch baru:
   ```bash
   git checkout -b feature/nama-fitur
   ```
4. Ubah kode dan tambahkan dokumentasi bila perlu.
5. Commit perubahan:
   ```bash
   git add .
   git commit -m "Menambahkan fitur/fix ..."
   ```
6. Push ke GitHub:
   ```bash
   git push origin feature/nama-fitur
   ```
7. Buka Pull Request dan jelaskan perubahan Anda.

## Aturan umum

- Jaga nama file, struktur, dan style.
- Uji script (`python Ghost-Web-Scanner.py`) sebelum PR.
- Sertakan penjelasan perbaikan/fitur di deskripsi.
- Jangan tambahkan kredensial atau data sensitif.
